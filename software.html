Software

Completed open-source software packages and accompanying manuscripts.

Machine Learning

  • Single-cell-RNA transcriptome disease classifier — A neural network classifier for transcriptomics data with LIME interpretability to extract gene signatures most important for disease classification.
    Software | Preprint: Parkinson diseae signatures extracted from post mortem brain
  • Ensemblex — A weighted-consensus genetic demultiplexing method to accurately identify subjects in pooled single cell sequencing data with mixes of > 8 samples.
    Software | Manuscript: Method validation and application to ADHD patients
  • Microscopy Image Disease Classifier — Deep learning CNN model for classifying disease status from microscopy images.
    Software | Preprint: Disease classification of Parkinson disease cells and healthy controls.

Analysis Workflows

  • Auto-qPCR — Automated analysis of DNA & RNA qPCR data processing both absolute and relative sequencing and providing processed data, statistics and barcharts.
    Software | Manuscript: How the tool works and the biological considerations.
  • scRNAbox — End-to-end scRNAseq analysis workflow for data processing from FASTQ to differential gene expression on cloud computing clusters.
    Software | Manuscript: The complete workflow with biological explanations for processing decisions.
  • CelltypeR — Flow cytometry data analysis and visualization toolkit to identify and quantify cell types in complex tissues.
    Software | Manuscript: Quantification of cell types midbrain organoids across genetic background and time.
  • AXION Spike Analysis — Burst and network activity analysis for MEA recordings.
    Software | Preprint: Analysis of bursting patterns in midbrain organoids

Microscopy

  • Automated Cilia Measurements — Quantification of cilia features in microscopy images.
    Software
  • HistQ — Automated quantitative analysis of tissue section histology images.
    Software | Manuscript: Cell type and protein expression in mouse brain sections.
  • CellQ — Quantitative analysis of neuronal cell culture images.
    Software | Manuscript: Cell morphology and protein expression in stem cell-derived cultures.
  • OrgM — Organoid size measurement from imaging data.
    Software
  • OrgQ — Cell type, morphology and protein marker expression level quantification in organoids.
    Software | Cell type markers and synuclein expression in midbrain organoids.
