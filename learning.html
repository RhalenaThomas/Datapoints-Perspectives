Workshops & Workbooks

These workshops were developed for hands-on training sessions, with code and instructions on how to get the example data on GitHub.

  • The Neuro Single Cell Workshops — R workbooks for scRNA analysis from read counts (CellRanger/Trailmaker), dimensional reduction, data integration, clustering and annotation. Includes 2023 (Seurat v4) and 2024–2025 (Seurat v5).
  • McGill Initiative in Computational Medicine Single Cell Workshop — R workbook for scRNA-seq analysis with example data.
