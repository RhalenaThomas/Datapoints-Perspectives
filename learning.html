Workshops

These workshops were developed for hands-on training sessions, with code and instructions on how to get the example data on GitHub.

  • The Neuro Single Cell Workshops — R workbooks with code for scRNA analysis from read count outputs (CellRanger or Trailmaker outputs), dimensional reduction, data integration,clustering and annotation. Workbooks from 2023 (Seurat V4), 2024 & 2025 (Seurat V5) are included.
  • McGill Initiative in Computational Medicine Single Cell Workshop — R workbook for scRNAseq analysis with example data.
